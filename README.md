📌 Objetivo del proyecto

Analizar el comportamiento de usuarios de ConectaTel para identificar patrones de uso, segmentar clientes y generar recomendaciones que optimicen la oferta de planes.

📂 Datasets utilizados
user_profile: información demográfica y plan contratado
usage: registros de uso (llamadas y mensajes)
everpeak_retail (apoyo): validación y práctica de limpieza
🔍 Etapas del análisis
Limpieza y validación de datos
Tratamiento de valores faltantes y outliers
Análisis exploratorio (EDA)
Segmentación de usuarios
Generación de insights y recomendaciones
