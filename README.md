📊 Análisis de Datos – ConectaTel

Este proyecto presenta un análisis exploratorio y proceso de limpieza de datos para una compañía de telecomunicaciones, con el objetivo de identificar patrones de uso, segmentar clientes y generar recomendaciones de negocio basadas en evidencia.

🔧 Procesamiento de datos

Se realizó una limpieza integral del dataset que incluyó:

Conversión y validación de variables de tipo fecha
Identificación y tratamiento de valores faltantes e inválidos
Manejo de valores atípicos mediante el método del rango intercuartílico (IQR)
Estandarización de variables categóricas
Eliminación o ajuste de registros inconsistentes (como fechas futuras)
📈 Análisis exploratorio

Se analizaron variables clave como:

Edad de los usuarios
Número de llamadas y mensajes
Duración de llamadas
Tipo de plan contratado

Se utilizaron estadísticas descriptivas, distribuciones y visualizaciones para identificar patrones y comportamientos relevantes.

👥 Segmentación de usuarios

Se construyeron segmentos de clientes basados en su nivel de uso:

Bajo uso
Uso medio
Alto uso

Esto permitió entender diferencias en comportamiento y detectar usuarios de alto valor.

🧠 Insights clave
El comportamiento de uso es el principal diferenciador entre usuarios
Existen usuarios intensivos con alto consumo de minutos
La edad no es un factor determinante en la elección del plan
El plan premium está asociado a un mayor nivel de uso
🚀 Recomendaciones
Crear planes diferenciados para usuarios intensivos
Implementar estrategias de upselling hacia planes premium
Mejorar la calidad y consistencia de los datos
